Overview

This class allows you to quickly create a new JWT application.

Examples:

application = Heimdallr::CreateApplication.new(
   name: 'My Little Pony',
   scopes: %w[unicorn:create unicorn:update unicorn:hug unicorn:ride],
   algorithm: 'RS256',
   ip: request.remote_ip
 ).call
